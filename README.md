# Default Google Groups to Send As Group

A Google Chrome extension that automatically selects "Send on behalf of <Your Group>" when using Google Groups.

This makes it so that using Google Groups as a collaborative inbox is easier. A customer can simply hit reply and the reply will go to the group.

# Install
Get it for free from the [Chrome Web Store](https://chrome.google.com/webstore/detail/send-as-group-default/hlddnlceihdeefkfacopnokmdlaapnho)
